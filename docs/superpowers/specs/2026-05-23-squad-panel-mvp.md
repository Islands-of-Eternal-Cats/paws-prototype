# PAWS: Squad Panel MVP — Design Spec

> Дата: 2026-05-23
> Статус: Утверждён
> Следующий шаг: writing-plans

## 1. Цель

Создать работающий UI-прототип главного экрана игры PAWS — Squad Panel. Статические данные, React + Vite, компонентная архитектура. "Hello world" в формате игрового интерфейса.

## 2. Стек

- **React 18** + **Vite** (TypeScript)
- Чистый CSS (CSS Modules или inline styles)
- Никаких внешних UI-библиотек
- Mock-данные в отдельном файле

## 3. Состав прототипа

### 3.1. Данные

Три предустановленных отряда:
- **KOBRA-1** — ASSAULT (readiness 84%)
- **MULE-3** — SALVAGE (readiness 62%)
- **GHOST-7** — RECON (readiness 91%)

Каждый отряд содержит:
- id, name, role
- readiness (процент)
- size (current/max)
- supply status по 4 ресурсам (ammo, fuel, medkits, materials) со статусами ok/low/missing
- 4 оператора (класс, оружие, утилита)
- squad cargo (общие ресурсы с лимитом)

### 3.2. Компонентная структура

```
src/
  App.tsx                    — корневой компонент, состояние выбранного отряда
  App.css                    — глобальные стили (тема)
  data/
    mockData.ts              — статические данные отрядов
  components/
    SquadSelector.tsx        — переключение между отрядами (табы)
    SquadHeader.tsx          — название, роль, численность
    ReadinessBar.tsx         — прогресс-бар + процент
    SupplyStatus.tsx         — список ресурсов со статусами ✓/⚠/✗
    ActionButtons.tsx        — кнопки RESUPPLY и AUTO
    OperatorCard.tsx         — портрет оператора с классом и снаряжением
    SquadCargo.tsx           — строка общих ресурсов отряда
```

### 3.3. Визуальные принципы

- Тёмная тема (background #0f0f1a, карточки #1a1a2e)
- Моноширинный шрифт для данных
- Цветовые индикаторы: зелёный (ok), жёлтый (low), красный (missing)
- Readiness bar с градиентом (жёлтый/золотой)
- 4 колонки для портретов операторов
- Никаких анимаций — статичный прототип

## 4. Что НЕ входит в MVP

- Бэкенд / логика
- Редактор шаблонов
- Кликабельность кнопок (только визуал)
- Анимации
- Адаптивность (desktop-only)
- Данные о FOB, миссиях, складе базы

## 5. Критерии готовности

1. Приложение запускается через `npm run dev`
2. Отображается Squad Panel в соответствии с макетом
3. Можно переключаться между 3 отрядами
4. Данные статические, без запросов
5. Код разбит на компоненты согласно структуре выше
